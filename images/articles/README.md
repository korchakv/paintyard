# Зображення статей

Ця папка призначена для зберігання зображень статей.

## Структура файлів

Зображення називаються за ID статті:
- `1.jpg` - зображення для статті з ID 1
- `2.png` - зображення для статті з ID 2
- `3.jpg` - зображення для статті з ID 3
- тощо...

## Використання

1. В адмін-панелі кожна стаття має свій ID
2. Завантажте зображення статті
3. Перейменуйте файл на [ID статті].jpg або [ID статті].png
4. Завантажте в цю папку на GitHub
5. В адмінці вкажіть URL: `images/articles/[ID].jpg`

## Рекомендації

- Формат: JPG або PNG
- Розмір: 1200x600px рекомендовано
- Розмір файлу: до 400KB
