# Зображення товарів

Ця папка призначена для зберігання зображень товарів.

## Структура файлів

Зображення називаються за ID товару:
- `1.jpg` - зображення для товару з ID 1
- `2.png` - зображення для товару з ID 2
- `3.jpg` - зображення для товару з ID 3
- тощо...

## Використання

1. В адмін-панелі кожен товар має свій ID
2. Завантажте зображення товару
3. Перейменуйте файл на [ID товару].jpg або [ID товару].png
4. Завантажте в цю папку на GitHub
5. В адмінці вкажіть URL: `images/products/[ID].jpg`

## Рекомендації

- Формат: JPG або PNG
- Розмір: 800x800px рекомендовано
- Розмір файлу: до 300KB
