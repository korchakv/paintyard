# Папка для зображень брендів

Ця папка містить логотипи та зображення брендів, які відображаються на сайті.

## Правила іменування файлів

Кожне зображення бренду повинно мати назву відповідно до ID бренду:
- `1.jpg` - зображення для бренду з ID 1
- `2.jpg` - зображення для бренду з ID 2
- `3.png` - зображення для бренду з ID 3
- і так далі...

## Підтримувані формати

- **JPG/JPEG** - рекомендовано для логотипів та фото
- **PNG** - для зображень з прозорістю
- **WebP** - сучасний формат
- **SVG** - для векторних логотипів

## Рекомендовані розміри

- Мінімум: 500x500 пікселів
- Рекомендовано: 800x800 пікселів або більше
- Формат: квадратний (1:1) або горизонтальний (16:9)

## Як додати зображення бренду

1. У адмін-панелі подивіться ID бренду
2. Перейменуйте ваш файл на `[ID].jpg` (наприклад, `3.jpg`)
3. Завантажте файл у цю папку через GitHub:
   - Натисніть "Add file" → "Upload files"
   - Перетягніть файл
   - Натисніть "Commit changes"
4. В адмін-панелі введіть URL: `images/brands/3.jpg`

## Приклад

Якщо у вас є бренд "Caparol" з ID 1:
- Назва файлу: `1.jpg`
- URL в адмін-панелі: `images/brands/1.jpg`
