# Інструкція: Створення og-image.jpg для соціальних мереж

## Проблема
SVG формат (og-image.svg) не підтримується Facebook, Twitter та іншими соціальними мережами.

## Рішення
Потрібно створити og-image.jpg з розміром 1200x630px

## Технічні вимоги:
- Формат: JPG або PNG
- Розмір: 1200 x 630 пікселів (співвідношення 1.91:1)
- Максимальний розмір файлу: до 1 MB
- Шлях: `/images/og-image.jpg`

## Що має бути на зображенні:
- Логотип Paintyard
- Текст: "Малярний Двір (Paintyard)"
- Підпис: "Магазин фарб в Івано-Франківську"
- Іконки або назви брендів: Teknos, Tikkurila, Aura, Rilak
- Телефон: (063) 770-24-57
- Адреса: вул. Вовчинецька 191

## Після створення:
1. Зберегти файл як `og-image.jpg` в папку `/images/`
2. Видалити старий файл `og-image.svg` (якщо він є)
3. Перевірити відображення через Facebook Sharing Debugger: https://developers.facebook.com/tools/debug/

## Онлайн інструменти для створення:
- Canva.com (шаблони Open Graph)
- Figma
- Adobe Photoshop/Illustrator
- Photopea.com (безкоштовний онлайн редактор)
